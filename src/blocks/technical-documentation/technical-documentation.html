Исправность оборудования, необходимого для изготовления металлоконструкций, проверяется в соответствии с графиком планово-предупредительного ремонта. За данную проверку отвечают: Служба механиков – 4 человека Служба энергетиков – 3 человека Освидетельствование кранов происходит 1 раз в год при осуществлении производственно-технического контроля
